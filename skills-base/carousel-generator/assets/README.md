# Assets

Put SVG / PNG / JPG files here to reference them from slides as `{{asset:filename.ext}}`.

Example: if you drop `rocket.svg` into this folder, you can use it in a slide's `body` HTML as:

```html
<img src="{{asset:rocket.svg}}" style="width:120px;">
```

The renderer inlines the asset as a data URI so it travels with the HTML — no external file refs in the final PNG.

## Where to get free icons

- **Heroicons** — https://heroicons.com/
- **Lucide** — https://lucide.dev/
- **Feather** — https://feathericons.com/
- **Phosphor** — https://phosphoricons.com/
- **Tabler** — https://tabler.io/icons

Download as SVG, drop here, reference by filename.
