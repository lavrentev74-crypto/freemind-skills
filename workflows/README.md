# n8n Воркфлоу

Готовые воркфлоу для импорта в n8n.

## Как импортировать

1. Открой n8n → Workflows → кнопка `+`
2. `Import from file` → выбери нужный JSON
3. Настрой credentials (токены, ключи) → Activate

## Категории

| Папка | Что внутри |
|-------|-----------|
| `telegram-bots/` | Боты с меню, квалификацией, рассылками |
| `content/` | RSS → AI → публикация в Telegram |
| `crm/` | Обработка заявок, уведомления, запись в БД |

## Скоро

Воркфлоу добавляются по мере выхода уроков клуба.
