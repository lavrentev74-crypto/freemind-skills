# FreeMind Skills

Скиллы, промпты и воркфлоу для Claude Code, Codex и ChatGPT.

Этот репо — **витрина**. Полные материалы только для участников клуба [@free_mind_rus](https://t.me/free_mind_rus).

## Что здесь есть (бесплатно)

| Раздел | Что внутри |
|--------|-----------|
| `skills-base/` | 32 базовых скилла |

## Установка

```bash
git clone https://github.com/lavrentev74-crypto/freemind-skills
cd freemind-skills
bash install.sh
```

Скрипт спросит что установить и покажет список всех скиллов с описанием пользы.
