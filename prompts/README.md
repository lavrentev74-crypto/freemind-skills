# FreeMind Prompts

Готовые промпты под разные задачи. Копируй и используй.

| Папка | Что внутри |
|-------|-----------|
| `content/` | Написание постов, текстов, контент-планы |
| `research/` | Анализ, стратегия, исследования |
| `audience/` | Целевая аудитория, сегментация, персона покупателя |
| `bots/` | Промпты для Telegram-ботов и AI-агентов |
